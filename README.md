## /goal 命令

```text
/goal 生成一个中文网页《Stripe 独立创业者产品学习地图》。

核心目标：
把文章里提到的 indie hackers / solo founders / 小团队产品，拆成读者能学习和行动的出海产品地图，而不是主要做事实核验。

页面必须包含：
1. 读者能学到什么：总结 8-10 条可迁移的产品/分发/定价/交付规律。
2. 优先研究样本：挑 10-12 个最值得点开的产品，说明应该看它的落地页、定价、获客入口、产品结构还是生态位。
3. 产品链接库：按人物列出产品链接，尽量给官网；没有官网时给创始人页、App Store、GitHub 或资料页。
4. 出海路线：目录/数据库、平台增强、AI 工作流、职业身份、分发工具、模板/工具包。
5. 行动清单：用 90 天节奏指导读者从选题、落地页、收费 MVP、渠道验证到复盘。

页面语气：
中文，直接、锋利、可执行。不要把重点放在“这是否为 Stripe 官方名单”，只在阅读说明里提醒：名单是公开转述线索，收入数字会变，很多案例不是严格一人公司。

设计要求：
像一张商业情报地图和产品拆解手册。第一屏要让读者知道：点开这些产品、拆它们的结构，比羡慕它们的收入更重要。

技术要求：
单页静态 HTML/CSS/JS，可直接打开，可部署 GitHub Pages。产品链接支持搜索和类型筛选。
```
